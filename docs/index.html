💎 全面能力

🤖

真·编码 Agent

Claude 驱动的工具调用循环（最多 200 轮），自己决定查资料、定位代码、精确改文件、跑测试。

✂️

精确编辑

只改需要改的部分，不整文件覆盖——带上下文的字符串替换，像 Claude Code 一样改代码。

🔍

代码搜索

正则搜索代码库（grep）+ Glob 文件查找——先搜后改，精确定位。

🌐

浏览器自动化

内置 Puppeteer——自动打开浏览器、点击、输入、截图、提取内容。答题/测试/数据抓取全搞定。

📹

屏幕录制

手动触发录屏，记录 agent 操作全过程，保存为 MP4。需 ffmpeg。

🛡️

智能信任分级

三级风险：safe(直接放行) / moderate(默认放行) / dangerous(弹窗确认)。可一键切全自动。

🔌

多模型可选

Claude（Anthropic）/ OpenAI / DeepSeek / Kimi / 豆包 / 本地 Ollama——设置里随时切换。

💬

流式打字机 + Markdown

实时流式输出 + 代码高亮 + 一键复制 + 思考过程可折叠。

⌨️

全局快捷键唤出

Ctrl/⌘ + Shift + 空格，任何时候都能唤出助手。

⬇️ 安装指南

1

安装 Node.js（只需一次）

去 nodejs.org，点左边绿色的大按钮下载 LTS 版（.pkg 安装包）。

下载后双击 .pkg 文件 → 一路点「继续」→ 输入密码 → 完成。

或者用 Homebrew：brew install node

2

下载项目

点击下面按钮下载 ZIP，然后双击解压：

⬇️ 下载 ai-floating-assistant.zip

解压后把 ai-floating-assistant-main 文件夹拖到桌面，重命名为 ai-floating-assistant

3

打开终端

按 ⌘ Command + 空格，输入 Terminal，回车。

4

安装依赖 & 启动

cd ~/Desktop/ai-floating-assistant-main
# 如果文件夹名是 ai-floating-assistant 就改成：
# cd ~/Desktop/ai-floating-assistant

npm install

安装需要 1-2 分钟，看到进度条走完就行。

npm start

✅ 屏幕右上角出现蓝紫色悬浮球🎉 点击它 → ⚙ 设置 → 填 API Key → 开始用

5

可选：安装浏览器 + ffmpeg

浏览器自动化需要 Chrome，录屏需要 ffmpeg。不装也能用核心编码功能。

brew install --cask google-chrome
brew install ffmpeg
1

安装 Node.js（只需一次）

去 nodejs.org，点左边绿色大按钮下载 LTS 版（.msi 安装包）。

下载后双击 .msi → 一路点 Next → 勾选 "Automatically install the necessary tools" → Install → 完成。

2

下载项目

点击下面按钮下载 ZIP：

⬇️ 下载 ai-floating-assistant.zip

下载完成后：右键 zip 文件 → 全部解压 → 把解压出的文件夹拖到桌面。
把文件夹重命名为 ai-floating-assistant

3

打开 PowerShell

按 Win + X → 选「Windows PowerShell」或「终端」。

或者按 Win 键，输入 powershell，回车。

4

安装依赖 & 启动

cd $env:USERPROFILE\Desktop\ai-floating-assistant-main
# 如果文件夹名是 ai-floating-assistant 就改成：
# cd $env:USERPROFILE\Desktop\ai-floating-assistant

npm install

安装需要 2-3 分钟，等进度条走完。

npm start

✅ 屏幕右上角出现蓝紫色悬浮球🎉 点击它 → ⚙ 设置 → 填 API Key → 开始用

5

可选：安装浏览器 + ffmpeg

Windows 自带 Edge 浏览器，浏览器自动化自动检测可用。ffmpeg 只有录屏需要。

Chrome 去 google.com/chrome 下载安装。
ffmpeg 去 ffmpeg.org 下载 → 解压 → 把 bin 目录加到系统 PATH。

📱 手机能装吗？

📲

手机上可以浏览、下载，但不能直接运行

这是个 Electron 桌面应用，需要电脑（macOS / Windows / Linux）才能运行。

但你可以：

  • 在手机上打开这个网站，下载 ZIP 文件传到电脑
  • 📋 复制安装命令发给电脑（微信/Telegram/AirDrop 等）
  • 🔑 手机上打开 console.anthropic.com 注册获取 API Key
  • 浏览功能说明和工具清单

💡 Tips：把这段命令发到电脑上：
git clone https://github.com/swordzlwfyd-hash/ai-floating-assistant.git && cd ai-floating-assistant && npm install && npm start

🔑 获取 API Key

Claude（最推荐）

最强的编码能力 + 联网搜索。去 console.anthropic.com 注册，创建 API Key。

DeepSeek（最便宜）

性价比最高。去 platform.deepseek.com 注册，充值 10 元够用很久。

Gemini（免费额度）

谷歌 AI，有免费额度。去 aistudio.google.com 获取 API Key。

Kimi / 豆包

国内厂商，速度快。Kimi 去 platform.moonshot.cn，豆包去火山引擎控制台。

本地 Ollama（完全免费）

装 Ollama 后用本地模型，不上传数据。选 OpenAI 兼容模式，URL 填 http://localhost:11434。

没有 Key？先试用

选 OpenAI 兼容 → Base URL 填 http://localhost:11434 → Model 填 llama3.1（需先装 Ollama）。
